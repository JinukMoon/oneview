Why OneView

포맷마다 앱을 찾지 마세요

하나의 진입점에서 대부분을 직접 렌더링하고, 직접 못 그리는 형식만 가장 잘 여는 앱으로 넘깁니다.

자체 렌더러 · Product

PowerPoint를 라이브러리 없이 직접 그립니다

OOXML을 직접 파싱하는 경량 캔버스 렌더러로 배경·도형·표·그룹, 그리고 마스터/레이아웃 디자인까지 그립니다. 확대하면 고해상도로 다시 그려 선명하게 유지됩니다.

EMU 좌표 정밀 변환 · 5단계 서식 상속 · 슬라이드 세로 스크롤

Rare · 한글

HWP / HWPX 인앱 렌더

휴대폰에서 멀쩡히 볼 무료 수단이 거의 없던 한글 문서를 WASM 렌더러로 화면에 그대로 띄웁니다.

Sharp · PDF

확대할수록 선명한 PDF

보는 배율에 맞춰 페이지를 다시 렌더링합니다. 한글 폰트와 세로·가로 혼합 문서도 각자 비율을 유지합니다.

Formats

지원 형식

대부분은 인앱에서 직접 렌더링합니다. 나머지는 설치된 앱으로 자동 전달합니다.

PDF
인앱 렌더 · 확대 시 재렌더로 선명 (pdf.js)
한글 HWP / HWPX
인앱 시각 렌더 (WASM)
Word .docx
표·스타일 유지 인앱 렌더
Excel .xlsx / .xls
시트 탭으로 전환하며 표 렌더
PowerPoint .pptx
자체 OOXML 캔버스 렌더러
이미지
jpg · png · gif · webp · svg 인앱 렌더
텍스트
txt · csv · md · json · xml, 한글 인코딩 자동 감지
.doc · .ppt · .rtf
가장 잘 여는 앱으로 자동 전달
HEIC / HEIF
사진 앱으로 전달

Platforms

웹앱과 Android 앱, 같은 코드로

브라우저에서 바로 쓰거나, 앱으로 설치하세요. 문서 렌더링 엔진은 두 곳에서 동일하게 동작합니다.

웹앱 · PWA

설치 없이 브라우저에서 즉시. 안드로이드 · iOS 모두.

  • ✓파일을 골라 PDF·한글·Office·이미지 열람
  • ✓홈 화면에 추가하면 앱처럼 실행 · 오프라인 동작
  • ✓공유는 Web Share, 미지원 시 다운로드로 폴백
  • —카톡·메일에서 자동으로 열기 (웹 표준 제약)

Android 앱 · APK

네이티브로 완전한 연동. 오프라인 · 무광고.

  • ✓모든 인앱 뷰어 (PDF·한글·Office·이미지)
  • ✓카톡·메일에서 파일을 누르면 후보로 등장
  • ✓하드웨어 뒤로가기 · 외부 앱으로 공유
  • ✓서버 없이 오프라인, 파일은 폰 밖으로 안 나감

지금 열어보세요.

설치도, 로그인도 필요 없습니다. 링크 하나면 됩니다.
